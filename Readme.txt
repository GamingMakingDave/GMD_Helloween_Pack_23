To install the script and the map, you need to drag the resource onto your server and add the following lines to your server.cfg:

ensure GMD_Halloween
ensure GMD_Halloween_Map

Additionally, you must enable the latest game build to load all the latest clothing, vehicles, and props!!!

+set sv_enforceGameBuild 2944
